# 关于 FLM.OPM
   基于.net6.0的跨平台WPF快速开发框架，基于Avalonia和ReactiveUI, 使用MVVM 和Mediator设计模式。
   
## 支持平台

* `Windows 10+(主要)`
* `Linux`
* `macOS`
